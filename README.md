# Bota-AI
## Botasaurus driver based AI browser automation.

Uses the google genai Gemini 2.5. flash model to browse the web.


Actions:
- **click**: `driver.click(selector)`
- **type**: `driver.type(text, selector)`
- **get_text**: `driver.get_text(selector)`
- **stop_browsing**: Stops and returns result

## Setup
- Set your api key -> GOOGLE_API_KEY
- Configure MAX_STEPS
- Configure GOAL
